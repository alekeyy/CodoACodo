FORMULARIOS

Ahora veremos una forma de intercambiar informacion con el usuario mediante estas cajas de texto y sus botones.

NO PODEMOS PROCESAR LA INFORMACION VIA HTML, DEBEMOS RECURRIR A HERRAMIENTAS BACKEND

Nuevamente, volvamos a donde veniamos, podemos crear formularios via FORM

En este hay que especificar algunos atributos:

  • action: la accion que lleva a cabo, mediante
    • form action="correo.com" para enviar a direccion de correo
    • form action="direccion del archivo" para que lo procese un programa/script


  • method:El atributo que se encarga de especificar la forma en la que el formulario es enviado
    • method="post", el que usaremos hasta no ver backend
    • method="get", sorpresa


Ya dentro de form usaremos la etiqueta INPUT

Nuevamente, tenemos otros atributos para especificar:

  • type: Para ver que modalidad de elemento de formulario
    • type="text", escribir texto normal
    • type="password", otorga confidencialidad
    • type="radio", muestra una casilla pinchable en la pantalla, generalmente se pone de a varios para elegir entre uno, deben tener valor distinto.
    • type="checkbox", caja de validacion se puede activar o desactivar con solo clickearlo, similar a radio.
    • type="submit" value="mensaje de la caja a enviar", para enviar toda la informacion previamente elegida, se pone al final del formulario
    • type="reset" value="mensaje para borrar los cambios", sirve para borrar el formulario por completo para rehacerlo.
    • type="button", un boton sin uso por ahora, por ahora...

  • name: de gran importancia para identificarlo en el programa de procesamiento.
  • size: define el tama単o de la caja de texto, si se lo pasa se podra seguir escribiendo pero el texto se movera a la izquierda
  • maxlength: Indica el tama単o maximo del texto, una vez que se llegue a este no se permite seguir escribiendo.

    • size =/= maxlength

  • value: define el valor inicial del campo para darle una idea al usuario de con que rellenarlo.
  • textarea name="name" rows="filas" cols="columnas"> texto predefinido... >/textarea>

Alternativa para dar a elegir entre opciones:

  • Select name="nombre"
    • option (si ponemos selected se pondra esta por defecto)>opcion>/option>
    • >/select>
